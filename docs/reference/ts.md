

### Environment Setup

1. Install node.js
2. Install typescript package using `npm install -g typescript`
3. Install an IDE.



`tsc main.ts`

`node main.js`

`tsc main.ts --watch` 监听`main.ts`的改变，并实时编译



`tsc --init` 创建 tsconfig.json 文件
